# Nanogrid

A retro puzzle game with a fully-featured level editor.

More gameplay information on itch.io: https://0stam.itch.io/nanogrid

The game was created as part of a university project by a 7-person team of first-year computer science students.

## Features

- 16 built-in levels
- An interactive environment with multiple types of linked doors, buttons, teleports, etc.
- **The game environment is Turing-complete and was even used to implement an 8-bit adder**
- A level editor that lets you quickly build levels, link interactive objects, and test them
- Support for switching gameplay to a text-based UI, thanks to the modular architecture
